📋 Krok 1 — Utwórz Google Apps Script (backend)

1Otwórz script.google.com i kliknij Nowy projekt
2Usuń domyślny kod i wklej poniższy skrypt:
const SHEET_ID = SpreadsheetApp.create("LodzharniaDB").getId(); function doGet(e) { const ss = SpreadsheetApp.openById(SHEET_ID); const sheet = ss.getSheetByName(e.parameter.sheet) || ss.insertSheet(e.parameter.sheet); const data = sheet.getDataRange().getValues(); return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const ss = SpreadsheetApp.openById(SHEET_ID); const payload = JSON.parse(e.postData.contents); const sheet = ss.getSheetByName(payload.sheet) || ss.insertSheet(payload.sheet); if (payload.action === "write") { sheet.clearContents(); if (payload.data && payload.data.length > 0) { sheet.getRange(1, 1, payload.data.length, payload.data[0].length) .setValues(payload.data); } } return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
3Kliknij Wdróż → Nowe wdrożenie, wybierz typ Aplikacja internetowa
4Ustaw "Kto ma dostęp" na Wszyscy, kliknij Wdróż i skopiuj URL

ℹ️ Jak to działa?

Dane są przechowywane lokalnie w przeglądarce (localStorage) i automatycznie synchronizowane z Google Sheets co 5 minut lub przy każdym zapisie.

Google Sheets działa jako baza danych — możesz tam przeglądać i edytować dane jak w Excelu. Każda kategoria (dostawy, sprzedaż, pracownicy...) to osobna zakładka w arkuszu.

Aby udostępnić aplikację, po prostu podziel się plikiem HTML. Każdy użytkownik musi tylko raz wpisać URL swojego skryptu.
